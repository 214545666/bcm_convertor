### 简单介绍

这是个能在三大电脑平台上将Kitten3/4文件（.bcm）转换为App文件的应用，能够帮助你将自己在kitten3/4上的作品在Windows/Mac OS/Linux上快速制作成独立应用程序！

<br>

### 与官方比较优缺点

#### 优点：  
* 文件大小不受限制  
* 多平台支持  
* 无次数限制  
* 等待时间短
#### 缺点：
* 图标需要手动更换
* UI丑，时不时卡顿
* windows下bug最多（winrar）
